This assistant listens to voice commands, understands the intent using Naive Bayes and performs actions like:

• Play songs on YouTube when browser is open

• Tell current date and time

• Search Wikipedia and Google

• Open applications (Chrome, VS Code, etc.)

• Control system volume

• Handle follow-up commands like “tell more” about a topic (short-term memory)

🔧 Tech stack used:

Python, SpeechRecognition, pyttsx3, scikit-learn, Naive Bayes, CountVectorizer, Wikipedia API, pywhatkit, pyautogui

How to use:

*Just Install the required packages and you can run like a normal python code.

*You can give some commands like open youtube, play song open, vs code like that
